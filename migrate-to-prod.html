🚀 Migrar Dados para Produção

Copia os dados do DRE do IndexedDB local para o Redis de produção (Vercel).

⚠️ Atenção: Abra este arquivo a partir do servidor de desenvolvimento (http://localhost:5000 ou similar), não diretamente do sistema de arquivos, para que o IndexedDB da aplicação esteja acessível.
ℹ️ O que esse utilitário faz: Lê todas as chaves resumo_dre:* do IndexedDB local e envia para o Redis da produção usando o endpoint /api/kv/bulk-set.
